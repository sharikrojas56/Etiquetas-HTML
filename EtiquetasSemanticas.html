Etiquetas Semánticas

Aprende sobre las etiquetas semánticas en HTML y su importancia para la estructura y accesibilidad de las páginas web.

Introducción a las Etiquetas Semánticas

Las etiquetas semánticas en HTML son aquellas que describen claramente su significado tanto para los desarrolladores como para los navegadores y motores de búsqueda. Estas etiquetas ayudan a estructurar el contenido de una página web de manera lógica y accesible.

Tipos de Etiquetas Semánticas

Algunas de las etiquetas semánticas más comunes son:

Header

La etiqueta <header> se utiliza para definir la cabecera de una sección o página. Suele contener el título, logo, navegación u otros elementos introductorios.

¿Para qué se utiliza?

Se utiliza para agrupar contenido introductorio o de navegación, proporcionando una estructura clara y mejorando la accesibilidad.
