Texto

Aprende cómo se estructuran los textos en HTML mediante encabezados, párrafos y etiquetas de formato, fundamentales para organizar correctamente el contenido de una página web.

Importancia del Texto en HTML

El texto es la base de cualquier página web. A través del texto se transmite la información principal al usuario, como títulos, descripciones, instrucciones y contenido educativo. En HTML, el texto no se escribe simplemente para mostrarse en pantalla, sino que se organiza mediante etiquetas específicas que le dan estructura y significado. Esto permite:

  • Organizar la información de manera clara.
  • Facilitar la lectura.
  • Mejorar el posicionamiento en buscadores (SEO).
  • Hacer la página más accesible.
Por eso es fundamental usar correctamente las etiquetas de texto.

Párrafo

La etiqueta <p> se utiliza para definir párrafos en HTML.
Un párrafo representa un bloque de texto que desarrolla una idea específica dentro del contenido de la página.
Cada bloque de información debe ir dentro de su propia etiqueta <p>, lo que permite mantener el texto organizado y mejorar la legibilidad.
El navegador agrega automáticamente un espacio antes y después de cada párrafo para separar visualmente el contenido.

Texto en Énfasis: <em>

La etiqueta <em> se utiliza para dar énfasis a una palabra o frase dentro de un texto. Generalmente el navegador la muestra en cursiva, pero su función principal es indicar que esa parte del contenido debe leerse con mayor intención o relevancia. Se emplea cuando se quiere resaltar una idea importante dentro de un párrafo sin convertirla en información principal. No debe utilizarse únicamente por apariencia visual, sino cuando realmente se desea enfatizar un fragmento del texto.

Salto de Línea: <br>

La etiqueta <br> se utiliza para insertar un salto de línea dentro de un mismo párrafo sin crear un nuevo bloque de texto. Es útil cuando se necesita separar información corta como direcciones, poemas, datos personales o listas simples. A diferencia de la etiqueta <p>, el <br> no genera un espacio grande entre bloques, solo mueve el contenido a la siguiente línea. No se recomienda abusar de esta etiqueta para crear espacios visuales, ya que el diseño y los márgenes deben manejarse con CSS.

Línea Horizontal: <hr>

etiqueta <hr> se utiliza para insertar una línea horizontal que permite dividir visualmente el contenido dentro de una página web. Su función es marcar una separación temática entre secciones o bloques de información. No contiene texto y se representa como una línea divisoria.
